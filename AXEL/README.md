#Tondeuse

**tondeuseJs** User interface with AngularJS

**tondeuseTv** Broadcast data with websocket & akka actors

###Requirements

**tondeuseJs** 
nodejs, grunt

**tondeuseTv** 
playframework 2.2.1


###Quick start

**tondeuseJs** 

 $npm install
 
 $grunt dev

**tondeuseTv** 
$play run

**Go to** http://localhost:8888
